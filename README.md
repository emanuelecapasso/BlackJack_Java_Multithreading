# BlackJack_Java_Multithreading
Sviluppo del gioco del Blackjack in Java con utilizzo di multithreading. Ogni giocatore viene eseguito come un thread separato, permettendo al gioco di simulare partite simultanee e rendere l'esecuzione più realistica. Il programma gestisce la logica del gioco secondo le regole del Blackjack. classico
